# Terminal Casino

A fun, interactive text-based casino application built in Python! Test your luck with classic games like Slot Machines and Roulette directly from your terminal.

## Features
- **Slot Machine**: Spin the reels with emojis! Win big with 3 of a kind or get a multiplier for 2 of a kind.
- **Roulette**: Classic wheel spinning. Bet on your lucky numbers and see if you can beat the odds.
- **Bankroll Management**: Start with a fixed balance and track your winnings (or losses) across multiple games.

## Getting Started

### Prerequisites
- Python 3.x installed on your machine.

### How to Run
1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd terminal-casino
   ```
3. Run the main file:
   ```bash
   python main.py
   ```
